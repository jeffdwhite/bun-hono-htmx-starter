Routing

Routes are defined centrally and mounted as sub-apps.

Route Definition Pattern

export const featureRoutes = {
  paths: {
    list: () => "/items",
    item: (id: string) => `/items/${id}`,
  },
  routePatterns: {
    list: "/",
    item: "/:id",
  },
};

Benefits

  • Type-safe link generation
  • Centralized route management
  • Easy refactoring
